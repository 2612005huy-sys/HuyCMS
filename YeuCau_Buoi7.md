# Yêu cầu Buổi 7
**Trạng thái:** Đã hoàn thành
**Tên công việc:** Kết nối Frontend ReactJS với Backend ASP.NET Core Web API.
**Chi tiết triển khai:** Cấu hình CORS trên Backend, thiết lập Axios Client tập trung (axiosClient.js), xây dựng component hiển thị danh mục sản phẩm (CategoryProductList.jsx). Tự thực hiện bài tập mở rộng kết nối API danh sách sản phẩm (ProductList.jsx hiển thị Grid Card, định dạng VND) và tin tức (PostList.jsx hiển thị bài viết, định dạng ngày vi-VN).
